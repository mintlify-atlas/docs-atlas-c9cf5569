## What is E-Dokan?

E-Dokan is a full-featured e-commerce storefront application built with Next.js 13 App Router. It connects to an external REST API backend to display products, categories, and billboards — giving shoppers a seamless browsing and purchasing experience.

## Architecture overview

E-Dokan is a **frontend-only** storefront. All product data, categories, billboards, sizes, and colors are fetched from an external API specified by the `NEXT_PUBLIC_API_URL` environment variable. The app uses Next.js server components for data fetching and client components only where interactivity is required (cart, modals, filters).

<Note>
  E-Dokan requires a compatible backend API. The API URL is configured via `NEXT_PUBLIC_API_URL` in your `.env.local` file.
</Note>
